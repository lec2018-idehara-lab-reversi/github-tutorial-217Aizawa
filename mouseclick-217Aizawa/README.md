# 課題
## mouseClicked 関数の記述
mouseClicked() 関数の中で、組み込み変数 mouseX, mouseY を参照して、正しい場所の盤面を書き換えなさい。

ヒント：画面座標系 (0-639, 0-639) から、論理座標系 (1-8, 1-8) への変換を行う。一つのマスのサイズは CELLSIZE で定義されている。

## 手番の変更
一つ石を置くごとに、黒（１）と白（−１）が入れ替わるよう、手番の変更をプログラム上で記述しなさい。
